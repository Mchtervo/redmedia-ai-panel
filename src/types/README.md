# types/

Paylaşılan TypeScript tipleri.

- `database.ts` — Supabase şemasından üretilen/derlenen tipler (bkz. ilgili
  görev: "Supabase şemasından types/database.ts üret").
- İleride: `api.ts` (`ApiResponse<T>` zarfı — bkz.
  `.cursor/rules/01-code-quality.mdc`), ortak enum'lar.

Feature'a özel tipler burada değil, ilgili `features/<isim>/types.ts` içinde
kalır.
