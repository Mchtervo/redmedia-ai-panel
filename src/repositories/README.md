# repositories/

Yalnızca paylaşılan/generic repository yardımcıları (örn. sayfalama/filtre
helper'ları, ortak sorgu yardımcı fonksiyonları). Entity'ye özel
repository'ler (örn. `contacts.repository.ts`) burada değil, ilgili
`features/<isim>/repositories/` içinde durur (bkz.
`.cursor/rules/03-database.mdc`).

Henüz kod eklenmedi.
